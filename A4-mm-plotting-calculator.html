Assignment 4: JavaScript Calculator + Plot

This page is my Assignment 4 JavaScript calculator and plot using the Michaelis–Menten (MM) model. Enter seed values for Vmax and Km, set an S range (Smin, Smax, Step), and click Compute & Plot to graph the nonlinear curve v(S) = (Vmax·S)/(α·Km + S). You can optionally enable competitive inhibition (α = 1 + I/Ki) to see how it shifts the half-max point. The plot shows v on the Y-axis and S on the X-axis, marks the half-max point, lists computed values in a table, and lets you download them as CSV.

Half-max at S = Km (α=1). v = Vmax/2 there.

Plotting library: Plotly.js. Model: Michaelis–Menten with optional competitive inhibition.
