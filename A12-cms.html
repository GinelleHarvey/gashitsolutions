A12 — CMS Comparison (JSON Table)

Content Management Systems Table

This assignment compares several web content management systems (CMS) by loading their details from an external JSON file and rendering them into an interactive table using JavaScript. Each row shows the CMS name, support model, technology stack, major capabilities, limitations, and at least one example site that uses it.

Data source: A12-cms.json, loaded via fetch() and processed on the client side.
